# ChamaTrust Backend

Trust & Reputation Engine for ChamaConnect built on Stellar blockchain.

## Features
- Google OAuth login (Web2 UX)
- Automatic Stellar wallet creation
- Trust score calculation based on contribution/loan behavior
- Loan risk assessment (LOW/MEDIUM/HIGH)
- Dispute resolution system
- All transactions anchored on Stellar testnet
